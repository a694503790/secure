# 中间件安全

- 是否已关闭中间件的管理控制台（如是Weblogic/Websphere则填是并备注说明）
- 是否已开启中间件的记录访问日志
- 是否已配置为高可用模式（硬件如F5或软件如Keepalive或云负载均衡）
- 是否已设置中间件的httpd.conf、Server Root的所属用户所属组及权限（644）
- 是否已禁止中间件访问 Web 目录之外的任何文件
- 是否已禁止中间件目录列出
- 是否已设置错误页面重定向
- 是否已隐藏中间件的版本号
- 是否已关闭中间件的TRACE功能
- 是否已设置中间件禁止运行 CGI 程序
- 是否已设置中间件绑定监听地址为非0.0.0.0
